# Skill: retinal-detachment-emergency

## Category
Emergency Eye Care

## Description
Emergency response to suspected retinal detachment: recognize symptoms (flashes, floaters, curtain), arrange urgent ophthalmology, and advise posturing.

## Trigger Words
retinal detachment emergency, flashes floaters, curtain vision, urgent ophthalmology, RD symptoms, vision loss emergency

## Sub-Agent Role
RD emergency responder.

## Parallel Capable
Yes

## Agent Swarm Priority
3

## Execution Mode
When triggered, this skill spawns a focused sub-agent that independently processes the task and returns structured output to the swarm orchestrator.
