# Skill: lens-verification

## Category
Optical Dispensing

## Description
Verify finished spectacles against Rx: sphere, cylinder, axis, prism, segment height, PD. Use lensometer, and identify dispensing errors.

## Trigger Words
lens verification, Rx verification, lensometer check, dispensing error, spectacle check, final inspection

## Sub-Agent Role
Lens verification agent.

## Parallel Capable
Yes

## Agent Swarm Priority
3

## Execution Mode
When triggered, this skill spawns a focused sub-agent that independently processes the task and returns structured output to the swarm orchestrator.
