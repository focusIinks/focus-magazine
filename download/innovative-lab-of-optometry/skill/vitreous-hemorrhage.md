# Skill: vitreous-hemorrhage

## Category
Posterior Segment Disease

## Description
Assess vitreous hemorrhage: determine etiology (PVD, diabetes, trauma, neovascularization), B-scan if no view, and manage expectantly vs surgical referral.

## Trigger Words
vitreous hemorrhage, vitreous bleed, B-scan, vitreous opacity, no fundus view, vitreous clearing

## Sub-Agent Role
Vitreous hemorrhage assessor.

## Parallel Capable
Yes

## Agent Swarm Priority
3

## Execution Mode
When triggered, this skill spawns a focused sub-agent that independently processes the task and returns structured output to the swarm orchestrator.
