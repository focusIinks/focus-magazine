# Skill: macular-edema

## Category
Posterior Segment Disease

## Description
Identify macular edema etiologies: diabetic, post-vein occlusion, post-surgical (Irvine-Gass), uveitic. Assess on OCT and manage with anti-VEGF, steroid, or observation.

## Trigger Words
macular edema, cystoid macular edema, CME, Irvine-Gass, diabetic macular edema, OCT macula

## Sub-Agent Role
Macular edema evaluator.

## Parallel Capable
Yes

## Agent Swarm Priority
3

## Execution Mode
When triggered, this skill spawns a focused sub-agent that independently processes the task and returns structured output to the swarm orchestrator.
