## 3. CROSS-SITE ANALYSIS

### 3.1 Relationship
- **OptoBharat.com** is a product of **Envision Bharat**
- OptoBharat.com has a single product: OptoAI
- EnvisionBharat.com is the parent company showcasing 3 products: OptoAI, NexPOS, KhaoJi

### 3.2 Content Overlap
- Both sites have identical OptoAI pricing
- Both sites share the same Envision Bharat branding
- Both sites use the same Firebase/Cloudinary assets
- Both use dark theme with similar design language

### 3.3 Technical Stack Comparison
| Aspect | OptoBharat | EnvisionBharat |
|--------|-----------|----------------|
| Framework | React + Vite (bundled) | React 19 via CDN (esm.sh) |
| CSS | Tailwind (bundled) | Tailwind via CDN |
| Payment | Razorpay | None (only contact forms) |
| Sitemap | 1 URL only | 6 URLs |
| robots.txt | Present | Missing |
| JSON-LD | None | Organization schema |
| Structured Data | None | Basic |

### 3.4 Critical Findings Summary

1. **Razorpay Test Mode on Production (OptoBharat):** The Razorpay checkout script loads in "Test Mode" on the live site. While the "Test Mode" banner is hidden with `opacity: 0`, the Razorpay API is configured with `traffic_env=production` but the test mode span is present in the DOM.

2. **Missing Legal Pages (Both Sites):** Neither site has proper Terms of Service, Privacy Policy, Cookie Policy, or Refund Policy pages. EnvisionBharat's NexPOS page mentions non-refundable payments in an inline section but there's no standalone legal page.

3. **Our Story is a Dead Link (EnvisionBharat):** The "Our Story" button in the footer loads the homepage with no unique content.

4. **No robots.txt (EnvisionBharat):** The /robots.txt endpoint returns the homepage HTML instead.

5. **Production CDN Dependencies (EnvisionBharat):** Loading React and Tailwind from CDN in production is a reliability risk.

6. **Navigation Accessibility (EnvisionBharat):** Using `<button>` elements for navigation links instead of `<a>` tags is bad for both SEO and screen reader accessibility.

7. **Pricing Duplication:** OptoAI's Weekly (₹129/week) and Pro (₹299/month) plans have identical features, which may confuse customers.

8. **No Social Media Presence:** OptoBharat has zero social links. EnvisionBharat only has LinkedIn.

9. **No Blog/Resources:** Neither site has any content marketing — no blog, case studies, tutorials, or documentation.

10. **Only 1 Sitemap URL for OptoBharat:** The optobharat.com sitemap only indexes the homepage despite the site having multiple sections.
